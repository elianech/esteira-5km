# 🏃 Treino Esteira 5km — PWA

Aplicativo de treino na esteira com guia por voz, para correr 5km direto.

## Como instalar no celular

1. Acesse o link do GitHub Pages deste repositório
2. No Chrome (Android): toque nos 3 pontinhos → "Adicionar à tela inicial"
3. No Safari (iPhone): toque em Compartilhar → "Adicionar à Tela de Início"

## Funciona offline

Após a primeira abertura, o app funciona sem internet.

## Recursos

- 10 semanas de treino progressivo
- Voz em português anunciando cada fase
- Alertas a 30s e 10s do fim de cada etapa
- Timer regressivo por fase
- Barra de progresso geral
- Funciona com fone de ouvido Bluetooth
